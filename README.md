# Python File Organizer

This is a simple Python script that organizes files in a folder based on their file types.

## Features
- Automatically creates folders like Images, Documents, Videos, Audio, Archives
- Moves files into respective folders
- Keeps your directory clean and organized

## Modules Used
- os
- shutil

## How to Run
1. Download or clone this repository
2. Place the script inside the folder you want to organize
3. Run the script:
   python file_organizer.py

## Output
Before running the script: all files are in one folder  
After running the script: files are organized into separate folders
